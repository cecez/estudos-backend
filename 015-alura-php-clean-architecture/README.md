Value Objects
- se valores são iguais, referem-se ao mesmo objeto

Entidades
- possuem propriedade identificadora para distinguir

Serviços
- não se encaixam como entidades ou value objects
- subtipos:
  - Domain Services são classes que representam uma ação entre mais de uma entidade
  - Application Services / Use cases controlam o fluxo de alguma regra em nossa aplicação
  - Infrastructure Services são implementações de interfaces presentes nas camadas de domínio ou de aplicação
